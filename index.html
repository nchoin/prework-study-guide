HTML

  • The head element contains information about the webpage not visible to user
  • The body element represents what content is visible to the user
  • Elements = type of content on webpage
    • text
    • image
    • header
    • list
  • Some elements are used to divide and describe a page
    • Meta related = defines meta data like character set, description, keywords, author and viewoint
    • Content Divisions = line breaks or an actual dividing line
    • Semantic = name of the elementa describes its conents and role it plays
  • work together to build structure and provide framework that's easy for user to navigate the webpage
  • child elements=elements nested in other elements (parent elements)
  • terms elements and tags are often used interchangeable but they are DIFFERENT
    • element=entire from opening tag to the closing tag
    • tag= what's inside the angle brackets
  • when using heading tags h1, h2, h3, etc must use in consectutive order, it isn't assoicated with font display
  • if you want to have text wrap on the webpage you need to us paragragh tag

CSS

  • CSS=programming language to define how webpage content is presented to users
  • colors
  • aesthetics
  • visual layout
  • organization
  • create and define style Rules
  • 3 ways to style a page with CSS
    • Inline CSS - apply to specific tag, style element. direct, quick, permenant changes, must edit individually
    • Internal CSS Style Sheet - can embed a style in the head use styleproperties and values
    • External CSS Style Sheet - most common way to style a page - keeps CSS in a sep file which makes design changes easier. link to the file in head using a link rel="stylesheet" href="path to the sheet"
  • Selector=defins element or attribute to which rules or declarations apply
  • Declaration=property: value: example color: blue;
  • Declaration Block=group of declarations
  • if you want to apply a rule to ALL elements visible on a page use a WILD CARD symbol (*) in the selector space
  • can apply CSS to images too
    • use img selector and assign property and value to any image element
  • need to LINK the style sheet to the HTML file using a link element
    • add link element after the LAST meta element in HEAD but before the Title element
    • example of this is link rel="stylesheet" href="location path"
    • rel=attribute that means relationship b/t current doc and linked doc resource
    • href=attribute that specifies the location of the external source

Git

  • git status= check what branch we are currently on
  • git checkout -b branch-name= creates a new branch and switches to it
  • git add -A= add all changes
  • git commit -m " " what is in the quotes is your message to other developers
  • git pull origin main= pulling the source code from git hub because you want to be working the latest commits & changes
  • git push origin feature/starter code= push to yoru branch not the main branch. starter code is an example, you will use the name of the branch there
